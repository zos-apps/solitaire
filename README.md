## About

Classic Klondike solitaire card game.

## Features

- Standard Klondike rules
- Undo
- Auto-complete
- Statistics

## Installation

### From App Store
Open the **App Store** in zOS and search for "Solitaire".

### Via npm
```bash
npm install @zos-apps/solitaire
```

## Usage

- Drag cards to move
- Build up on foundations
- Click deck to draw

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+Z` | Undo |
| `N` | New game |
| `H` | Hint |


## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
